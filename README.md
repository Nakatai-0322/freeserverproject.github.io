# official-site
FREESERVERの公式サイトになる予定です

Pushすると自動でビルドされCloudflare Pagesで公開されます。

### デバッグ

#### Recommended VS Code extension
[Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

#### Requirements
- `Node.js` 16 or higher
- `pnpm` 7 or higher

```
npm run serve
```
